# Inventory Tracking
Inventory anaysis and management for a St. Louis area diaper bank distribution site.  

## Analysis Folder
Jupyter notebooks for analysis.
- **demand.ipynb** - Exploratory analysis of available data, focusing on demand, missing data, and inventory strategy.
- **missing.ipynb** - Totals for incorrect data entry at different distribution sites.

## Reports Folder
Documents summarizing analysis findings and suggesting next steps.
- **Diaper Storage Optimization** - Plan for reorganizing storage to better meet demand.
- **Diaper Stats Missing Values** - Summary of data entry issues and ideas for increasing entry accuracy.
- **Diaper Inventory Tracker Description** - Instructions on usage of an inventory management system created with Google Sheets/Apps Script.
